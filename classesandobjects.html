What are Classes and Objects in Java?

Classes and objects are fundamental concepts in object-oriented programming. A class is a blueprint for creating objects, which are instances of the class. Each object has its own set of attributes (also known as instance variables) and behaviors (also known as methods). In Java, a class is defined using the class keyword, followed by the class name and the class body enclosed in curly braces:

public class Person {
    // instance variables
    private String name;
    private int age;
    
    // constructor
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // methods
    public void sayHello() {
        System.out.println("Hello, my name is " + name + " and I am " + age + " years old.");
    }
}

In this example, we have defined a Person class with two instance variables (name and age), a constructor that takes in a name and an age, and a sayHello method that prints a greeting using the instance variables.

Creating Objects in Java

Once we have defined a class, we can create objects of that class using the new keyword:

Person alice = new Person("Alice", 25);
alice.sayHello();

Person bob = new Person("Bob", 30);
bob.sayHello();

In this example, we have created two Person objects, alice and bob, and called the sayHello method on each of them.
