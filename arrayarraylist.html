Arrays and ArrayLists

In Java, arrays and ArrayLists are used to store a collection of values of the same type. Here are some examples:

Arrays

Here's an example of how to declare and initialize an array:

int[] numbers = {1, 2, 3, 4, 5};

You can access individual elements of the array using their index:

int firstNumber = numbers[0]; // firstNumber = 1

ArrayLists

Here's an example of how to declare and initialize an ArrayList:

ArrayList<String> names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
names.add("Charlie");

You can access individual elements of the ArrayList using their index:

String secondName = names.get(1); // secondName = "Bob"
